# Marketplace 市场功能简要总结

Marketplace 是 HermesToken 的 AI API 能力市场。用户可以把自己的 AI API Key 托管到平台作为卖家供给，其他用户可以作为买家购买或调用这些供给。平台负责隐藏卖家 Key、代理请求、计费和收益结算。

## 核心功能

- 卖家托管 AI API Key，并设置可售模型、价格倍率、额度限制、时间限制和并发限制。
- 买家浏览市场供给，按供应商、模型、价格、额度、时长和并发等条件筛选。
- 买家可以购买固定订单，获得绑定到某个卖家 Key 的固定额度包。
- 买家也可以使用订单池，由系统自动选择合适的卖家 Key 完成请求。
- 市场能力可以绑定到用户 Token，使普通 OpenAI 兼容调用也能使用市场路由。
- 平台根据实际调用消耗计费，并为卖家生成收益和结算记录。

## 卖家侧能力

- 创建、编辑、上架、下架、启用、禁用托管 Key。
- 配置供应商、模型、价格倍率、额度、时长和并发。
- 检测托管 Key 的可用性和健康状态。
- 查看市场收益和结算流水。

## 买家侧能力

- 浏览和筛选市场供给。
- 购买固定订单并绑定到 Token。
- 查看固定订单的剩余额度、已用额度和状态。
- 使用订单池自动调用市场供给。
- 为 Token 保存订单池筛选条件。

## 管理员能力

- 开启或关闭 Marketplace。
- 配置允许托管的供应商类型。
- 配置平台市场费率。
- 配置固定订单购买额度范围和默认有效期。
- 配置卖家价格倍率和并发上限。

## 页面入口

Marketplace 页面主要包含：

- 订单列表
- 订单池
- 我的固定订单
- 卖家工作台
